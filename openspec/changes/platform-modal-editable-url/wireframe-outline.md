**头部元素：**

- 点击「+ 新增平台」→ 打开新增平台弹窗

### 2.2 平台卡片列表（折叠态）

每个平台卡片折叠态显示一行信息：

```
┌─────────────────────────────────────────────────────────────┐
│  🏛️ 中大平台     国家级 · 内网(VPN) · 数字验证码    [启用] [▼] │
│  ─────────────────────────────────────────────────────────── │
│  🏢 华东平台     省级 · 内网(VPN) · 字母+数字验证码   [启用] [▼] │
│  ─────────────────────────────────────────────────────────── │
│  ☁️ 数智大气     互联网 · 互联网 · 无需验证码          [启用] [▼] │
│  ─────────────────────────────────────────────────────────── │
│  🏙️ 乡镇数据平台  市级 · 互联网 · 数字验证码           [禁用] [删除] [▼] │
└─────────────────────────────────────────────────────────────┘
```

**卡片元素（折叠态）：**
- 左侧：平台图标 + 平台名称
- 中间：状态标签（已配置/未配置/已禁用）+ 级别 + 网络 + 验证码类型
- 右侧：启用开关 + 展开/折叠箭头

**状态标签规则：**
- `enabled && accounts.length > 0` → 绿色「已配置」
- `enabled && accounts.length === 0` → 橙色「未配置」
- `!enabled` → 灰色「已禁用」

**弹窗规格：**
- 宽度：`min(520px, calc(100vw - 32px))`
- 遮罩可关闭（非 loading 状态）
- 关闭动画：200ms 后触发 `close` 事件

**表单字段：**

| 字段 | 必填 | 类型 | 校验 | 说明 |
|------|------|------|------|------|
| 平台名称 | 是 | 文本 | 非空 | 唯一标识 |
| URL | 否 | 文本 | 无 | 平台地址 |
| 账号 | 否 | 文本 | 无 | 用户名 |
| 密码 | 否 | 密码 | 无 | 登录密码 |
| 别名 | 否 | 文本 | 无 | 账号别名 |
| 平台提示词 | 否 | 多行文本 | 最大1000字 | 平台上下文描述 |

**默认值：**
- 图标：`🔗`
- 级别：`自定义`
- 网络：`互联网`
- 验证码类型：`无需验证码`
- 启用状态：`true`

│  │                                                          │ │

│  └─────────────────────────────────────────────────────────┘ │
│  0/5000                                                       │
└─────────────────────────────────────────────────────────────┘
```

**选择器规格：**
- 网格布局：`repeat(auto-fill, minmax(200px, 1fr))`
- 卡片间距：12px
- 选中态：蓝色边框 + 浅蓝背景 + 右上角 ✓ 标记
- 悬停态：蓝色边框 + 阴影
- 禁用态：50% 透明度 + 不可点击

**自动填充逻辑：**
- 选中平台 → 自动将平台提示词追加到任务提示词
- 用户手动编辑提示词后 → 标记 `hasManualEdit = true`
- 取消选中时 → 若已手动编辑，提示「提示词已手动修改，取消选中不会自动更新」

## 8. URL 编辑交互

### 8.1 查看模式

```
地址：https://www.zjghj.com.cn/  [编辑]
```

- 双击 URL 文本 → 进入编辑模式
- 点击「编辑」按钮 → 进入编辑模式

### 8.2 编辑模式

```
地址：[https://www.zjghj.com.cn/        ] [保存] [取消]
```

- 回车键 → 保存
- 点击「保存」→ 更新 store 中的 URL
- 点击「取消」→ 恢复原值
- 输入框聚焦时自动全选

## 交互流程图

```
数据源平台设置页
    │
    ├─── 点击「+ 新增平台」───→ 新增平台弹窗
    │                              │
    │                              ├─── 填写表单
    │                              │
    │                              └─── 点击「添加」───→ store.addPlatform()
    │                                                         │
    │                                                         └─── 关闭弹窗
    │
    ├─── 点击平台卡片 ────→ 展开/折叠
    │
    ├─── 展开平台卡片
    │     │
    │     ├─── 点击「编辑」(URL) ────→ URL 编辑模式
    │     │                              │
    │     │                              ├─── 回车/保存 ────→ 更新 URL
    │     │                              │
    │     │                              └─── 取消 ────→ 恢复原值
    │     │
    │     ├─── 点击「+ 添加账号」───→ 账号管理弹窗
    │     │                              │
    │     │                              └─── 保存 ────→ store.addAccount()
    │     │

    │                                        │
    │                                        └─── 保存 ────→ store.updatePrompt()
    │
    └─── 点击「管理平台」(选择器中) ────→ 跳转到设置页

## 数据模型

```typescript
interface DataSourceAccount {
  id: string
  username: string
  password: string
  alias: string
  isDefault: boolean
  status: 'available' | 'pending' | 'error'
}

interface DataSourcePlatform {
  id: string
  name: string
  icon: string
  level: string        // 国家级/省级/市级/自定义
  network: string      // 内网(VPN)/互联网
  captchaType: string  // 数字验证码/字母+数字验证码/无需验证码
  url: string
  prompt: string
  enabled: boolean
  accounts: DataSourceAccount[]
  expanded: boolean
}
```
